# Yayasan Budi Rahayu Al Barokah

Website statis untuk Yayasan Yatim Piatu Budi Rahayu Al Barokah.

## Deployment ke Vercel

Untuk deploy sebagai static site:

- **Framework Preset**: Other
- **Build Command**: (kosong)
- **Install Command**: (kosong)
- **Output Directory**: ./

## Struktur Proyek

- `index.html`: Halaman utama
- `css/`: File CSS
- `js/`: File JavaScript
- `img/`: Gambar dan logo
- Halaman lainnya: berita.html, donasi.html, dll.

## Catatan

Proyek ini adalah static HTML/CSS/JS tanpa build system. Semua file siap deploy langsung.
